# Legenddary - AI-Powered eBook Creation Platform

## Original Problem Statement
Build a comprehensive, all-in-one AI-powered eBook creation platform that guides users from first draft to publication.

## Architecture
- **Backend**: FastAPI + MongoDB + Python
- **Frontend**: React + Tailwind CSS + Shadcn UI
- **AI Integration**: GPT-5.2 via Emergent LLM Key

## User Personas
- Authors and writers
- Self-publishers
- Content creators
- Small publishing houses

## Core Requirements (Static)
1. Rich Text Editor with formatting
2. AI Writing Assistant (content, footnotes, style)
3. Cover Designer
4. Signature Studio
5. Image Finder (Unsplash/Pexels)
6. Export (PDF, EPUB)
7. Publishing Guide
8. Marketing Tips
9. Project Management Dashboard
